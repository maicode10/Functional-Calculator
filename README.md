# Functional Calculator ➕➖✖️➗

A functional calculator implementation in Java.

## Description
This project contains a Java-based calculator that performs basic arithmetic operations using functional programming principles.

## Requirements
- Java JDK 8 or higher

## How to Compile
```bash
javac FunctionalCalculator.java
```

## How to Run
```bash
java FunctionalCalculator
```

## Features
- Basic arithmetic operations
- Functional programming approach
- Clean and efficient code structure

## Usage
Run the program and follow the on-screen instructions to perform calculations.
